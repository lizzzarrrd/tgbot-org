# tgbot-org
Цель:  создать телеграм-бота, который превращает пересланные из чатов и каналов сообщения в события календаря. Он автоматически извлекает дату, время, место и ссылку, показывает краткий превью и за одно нажатие создаёт файл .ics или добавляет событие в облачный календарь
